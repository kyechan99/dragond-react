# Dragond

## What is Dragond?

Dragond will **show you a helper device as you drag**.

## Install

```
npm i dragond-react
```

## How to use?

It's very very easy.

```
<DragondProvider>
    <Dragond>Lorem Ipsum.....</Dragond>
    <DragondDevice>
        Helper
    </DragondDevice>
</DragondProvider>
```

Feel free to **drag within the blue area**.

    <Dragond>Lorem Ipsum.....</Dragond>

        Helper
    </DragondDevice>
